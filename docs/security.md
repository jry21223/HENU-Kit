# 用户安全与隐私说明

HENU-Kit 是由学生自主维护的校园工具项目，非河南大学官方平台。使用任何工具前，请先确认它的来源、维护状态和适用范围。

## 不要提交或分享敏感信息

无论是在 Issue、交流群、截图还是公开仓库中，都不要发送：

- 校园账号密码
- 邮箱或短信验证码
- Token、Cookie、Session 或登录链接
- 完整学号、身份证号、手机号和家庭住址
- 支付信息、成绩名单或包含他人隐私的文件

需要截图反馈时，请先遮盖姓名、学号、邮箱和其他可识别信息。

## 使用自动化工具时

部分项目可能涉及校园网登录、图书馆预约或其他校园系统操作。请遵守学校系统规则，并注意：

- 只使用自己的账号，不共享账号或会话
- 不进行高频请求、批量抢占或绕过限制
- 遇到异常、封禁提示或系统维护时立即停止
- 不在公共电脑或不可信设备上保存登录信息
- 停止使用工具后，及时清理本地保存的敏感配置

## 资料与内容

上传、分享或下载资料时，应尊重版权和个人隐私：

- 不公开来源不明或明确禁止传播的资料
- 不上传包含学生个人信息、成绩或联系方式的文件
- 发现侵权、错误信息或隐私泄露时及时反馈

## 发现安全问题

发现账号风险、敏感信息泄露、异常登录或工具存在安全问题时：

1. 立即停止继续操作。
2. 修改相关账号密码并撤销可疑会话。
3. 不要在公开 Issue 中粘贴漏洞利用细节或真实凭据。
4. 通过维护者主页或交流群联系维护者，并只提供必要的脱敏信息。
