# 本科生范围审核

本目录用于记录本资料包的范围边界。主目录中的通知、竞赛和细则只有在官网正文或附件明确指向本科生时，才进入本科生索引。

## 明确不纳入本科生范围

以下文件正文明确面向研究生或研究生班集体，已从学院目录移入本区保留，不参与本科生资料索引：

- 文学院2026年研究生评先奖优细则公示
- 基础医学院2025年研究生学业奖学金评审细则
- 基础医学院2025年研究生先进集体及个人评审细则
- 中原发展研究院研究生学业奖学金评定细则
- 中原发展研究院研究生国家奖学金评定细则

## 仍需核验正文范围

软件学院2025年企业和校友奖学金拟获奖名单公示的官方详情页目前可访问，但它是结果公示而不是评定细则，涉及个人信息，因此只保留元数据，不进入本科生通用索引。

软件学院2026年三好学生等评选实施细则的原受限记录已由官网扫描PDF正文核验，旧记录移入“已由完整正文替代”区，当前可用版本在软件学院的01_细则目录中。

## 推免文件的边界

“推免研究生”不等于“研究生通知”。本包保留的推免文件只有在其推荐对象明确为应届本科毕业生时才收录，例如法学院2026届、人工智能学院2025届、软件学院历史版、纳米科学与材料工程学院历史版及校级本科生推免细则。已取得推免资格本科生申请研究生的学院接收细则另行标注，不能当作本科生奖助或评优通知。
