# 软件学院

- 分类：工程技术类
- 官方网站：https://software.henu.edu.cn
- 快照日期：2026-07-17

## 子目录

- `01_细则`：保研、综合测评、奖学金、评先奖优等正文或受限状态；
- `02_竞赛目录`：该学院适用或高度相关的竞赛目录；
- `03_竞赛通知`：已提取的具体竞赛通知正文；
- `04_来源与采集配置`：学院来源、采集器和Hook元数据。

## 状态说明

- `official_college_*`：学院官网直接发布或列入学院栏目；
- `official_all_*`：校级通知明确面向全校或所有专业；
- `discipline_relevance`：HENUKit根据竞赛内容和学院学科方向建立的推荐映射，不代表学院已经正式认定；
- `list_extracted`：只确认了官方标题和发布日期，不能视为完整通知。
