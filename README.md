## HENU-Kit 是什么

HENU-Kit 用来集中展示和连接由学生维护的校园工具与学习资源。

这个公开仓库只保留游客、用户和学生需要看到的内容，包括：

- 可用工具与访问入口
- 基本使用说明
- 面向用户的公开路线图
- 安全与隐私提醒
- 交流群与反馈渠道

内部开发计划、架构、API、数据库、部署、测试和团队会议文档不在本公开仓库维护。

## 怎么使用

1. 从上方入口选择需要的服务。
2. 第一次使用独立工具时，先阅读对应仓库的 README。
3. 涉及校园账号的工具，只在可信的本地环境中配置自己的账号。
4. 遇到异常、高频请求提示或学校系统维护时，立即停止自动化操作。
5. 不要在 Issue、群聊、截图或公开仓库中发送密码、验证码、Token 或 Cookie。

详细提醒见 [用户安全与隐私说明](./docs/security.md)。
